📊 BASIC USAGE
═══════════════════════════════════════════════════════════════════

Analysis Only:
  python3 main.py your_report.pdf

Analysis + Interactive Q&A:
  python3 main_with_qa.py your_report.pdf --interactive

Q&A Only (after analysis):
  python3 interactive_qa.py parsed_document.json analysis_results.json

View Demo (no API needed):
  open demo_dashboard.html
  python3 demo_qa.py

🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

"API key not found"
  → Check .env file has ANTHROPIC_API_KEY=sk-ant-...

"Credit balance too low"
  → Add credits at console.anthropic.com/settings/plans

"No document data"
  → Run analysis first: python3 main.py your_report.pdf

Q&A seems generic
  → Make sure you passed analysis_results.json for full context
